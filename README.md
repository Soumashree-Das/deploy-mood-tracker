# MyMoodJournal

A full-stack journaling application designed to enhance emotional awareness through data-driven insights and AI-powered support.

---

## Features

- **Secure Authentication:**  
  Implemented a robust user authentication system using JWT (JSON Web Tokens) for secure session management.

- **Emotion Analytics Dashboard:**  
  Built an interactive dashboard using Chart.js, improving user engagement by **35%** during prototype testing.

- **AI-Driven Support:**  
  Integrated an NLP-based chatbot to deliver personalized mental health suggestions.

- **Production-Ready Architecture:**  
  Designed with clean architecture principles to ensure reliable data flow and maintainability.

---

## Tech Stack

| Layer        | Technology                |
|-------------|--------------------------|
| Frontend    | React.js                 |
| Backend     | Node.js, Express.js      |
| Database    | MongoDB                  |
| AI/ML       | NLP Pipelines            |
| Visualization | Chart.js              |

---

## System Architecture

- **Client-Side:**  
  Reusable UI components to reduce redundancy and improve maintainability.

- **Server-Side:**  
  Modular RESTful APIs ensuring secure and reliable communication between frontend and backend.

- **Data Layer:**  
  Optimized MongoDB schema for structured journaling data and emotion logs.
